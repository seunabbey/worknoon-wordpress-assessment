# Short Answers

## 1. Difference Between Google Knowledge Graph and Google Knowledge Panel
The Google Knowledge Graph is Google’s database of entities and their relationships.
The Google Knowledge Panel is the visual information box displayed in search results using data from the Knowledge Graph.

In simple terms:
- Knowledge Graph = data system
- Knowledge Panel = visual display

## 2. How Google Determines Entity Identity
Google determines entity identity using:
- schema markup
- structured data
- consistent brand signals
- authoritative mentions
- social profiles
- backlinks
- website content
- external databases

Consistency across the web improves Google’s confidence in the entity.

## 3. When to Create Custom Post Types Instead of Pages
Custom Post Types should be created when content:
- requires unique structure
- needs custom fields
- will scale significantly
- requires filtering or categorization
- represents repeatable data

Examples:
- properties
- portfolios
- courses
- testimonials
- events

Regular pages are better for static content like:
- About page
- Contact page
- Privacy policy
